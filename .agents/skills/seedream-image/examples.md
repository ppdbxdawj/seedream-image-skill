## 2. Knowledge Cards & Infographics | 知识卡片类

### 仙侠风格卡片
```
生成一张仙侠古籍图鉴风格的卡片，向宗门弟子介绍[产品/概念]。
卡牌应采用水墨国风与工笔画相结合的画风，仙气缥缈，色彩淡雅。
布局上，中心是主体的精细插图，旁边配有竖排的详细楷体注释。
卡牌四周应有祥云或卷草纹的古典边框，背景素净，有大量留白，
整体质感如同一本传世秘籍中的一页。
```

### 现代信息图
```
创作一张现代极简信息图（Infographic），向都市白领解释[概念]。
图片应使用明亮、和谐的色块和简洁的扁平化图标，
信息布局遵循视觉引导，使用无衬线字体标注关键步骤，
有相对详细的文字介绍，整体风格干净有条理，
类似于高端商业分析报告中的图表。
```

### 游戏技能卡片
```
设计一张幻想RPG游戏中的技能卡片，用文字和图像解释"[概念]"。
有游戏化的技能名称，卡片上有酷炫图标、技能描述（用游戏化语言解释概念）、
消耗的"精力值"和冷却时间。整体是暗黑奇幻风格，带有发光的魔法符文边框。
```

### 塔罗牌风格
```
设计一张塔罗牌，用神秘学的象征手法诠释[概念]。
卡牌需要有经典的装饰性边框，中心是象征性的核心图像，
底部有卡牌名称、罗马数字和中文描述。
整体采用神秘、复古的版画风格，色彩象征意义丰富。
```

### 蓝图工程风格
```
生成一张技术蓝图风格的知识卡片，向工程师阐释[原理]。
图像使用经典蓝底白线，包含精确的几何线条、尺寸标注和剖面视图，
排版严谨，下方设有标题栏、图例和详细文字说明，呈现出专业工程图纸的质感。
```

## 3. PPT / Presentation Backgrounds | PPT 背景图类

### 磨砂玻璃风格（横向对比/并列）
```
一张电影海报级别的视觉作品，采用极简主义和空间主义风格。
三块大小不一的长方形磨砂玻璃板，优雅地悬浮在画面右半部分，
最大的玻璃板靠后，上面蚀刻着核心概念；前面两块较小的玻璃板前后错落。
玻璃材质半透明磨砂质感，边缘锐利。
一束强烈的聚光灯从左上方斜射而下，穿透玻璃，形成柔和光晕和清晰轮廓光。
背景是带有微弱颗粒感的哑光深灰色墙面。
所有视觉主体集中在画面右侧，为左侧留出大片干净背景区域，方便后期排版。
```

### 高质感金属风格（流程/递进）
```
数块大小不一的几何金属板，以解构主义方式悬浮在画面左侧。
整体材质为带细腻拉丝纹理的哑光黑钢，边缘锋利，反射冷峻光泽。
最大的三块垂直金属板上，分别用蚀刻工艺展示关键词。
一束戏剧性的聚光灯从画面外斜射而下，精准照亮重点元素，投下深邃阴影。
背景是纯粹的哑光深灰色。
整个视觉主体布置在画面左半部分，右侧形成大面积留白，供后期添加文字。
```

## 4. Avatars & Icons | 头像/图标类

### 玻璃质感图标
```
将这张照片的景色变成一个带有照片相关渐变色，
透明磨砂玻璃漫射材质，图标下方用艺术黑体写着跟图片景色相关的英文标题，
纯白背景的图标，C4D，OC渲染。
```

### 角色风格头像（参考图生成）
```
参考图1的风格和样式为图2的角色生成一个图标，
同时只要参考图2的头部位置，像一个头像图标，
图标下方的文字应该为"[名字]"
```

## 5. Character Consistency & Storyboards | 角色一致性/连续分镜

### 双角色场景（需上传参考图）
```
参考这两个角色的面部和发型，将其更改为[现代/古代/未来]装束，
生成6张连续的在[城市/场景]的分镜图，写实电影剧照风格，
需要在一个场景中，连续动作。
```

### 单角色多场景
```
保持这个角色的外貌特征，生成在[图书馆]、[咖啡厅]、[公园]三个场景中的图片，
角色服装风格统一，写实人物摄影风格。
```

## 7. Chinese Traditional & Mystical Art | 国潮/玄学类

### 护身符壁纸（宠物/IP）
```
以上传的[动物品种]的样貌和神态为原型，将其幻化成一只威风凛凛的山海经神兽。
神兽保留着宠物的特征，但体型巨大化，身披荧光彩绘的鳞甲与符文，体态夸张充满力量。

主体为以复古票据为原型，米黄底色，外围有繁复绿纹边框。
背景是充满张力的水墨风山海经世界，远山如黛，怪石嶙峋。

顶部和底部有重复英文"EVERYTHING GOES WELL"，中间"GOOD LUCK"，
中央用粗犷黑笔写着"[愿望]"，底部有英文"Wish you all the heavenly blessings"
与红色篆刻印章，国潮票据场景，复古花纹、书法狂草、篆刻印章与英文标语碰撞修饰。
```

## 8. Journal & Diary Style | 手帐/日记类

```
帮我生成一张图片，根据图片内容模拟在略带纹理的纸张上（米黄色）手写的今天的日记。
所有图片以拼贴画风格放在一页日记上，包含以下元素：

- 用手写字体描述今天做了什么，以及一两句标语
- 包含几张图片的简介，用红色笔迹圈出特别推荐的地点或活动
- 穿插一些与图片特色相关的简单涂鸦式小图
- 写着标题"[日期]""[城市] [天气]"
- 整体感觉像一份由热爱生活的作者制作的生动个人日记
```
