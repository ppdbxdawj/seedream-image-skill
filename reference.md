## 官方提示词指南（来自火山引擎 Seedream 3.0）

> 原文：https://www.volcengine.com/docs/82379/1795150

### 提示词秘籍

- 用简单直接的语言表达，短提示词也能发挥惊艳效果（长提示词 ≠ 好效果）
- 专业词汇推荐使用词源语言/英文，效果更准确
- 用**连贯自然语言**描述画面内容（主体+行为+环境），用**短词**描述画面美学（风格、色彩、光影、构图）
- 推荐写出图像用途和类型，例如：`PPT封面背景图`、`广告海报设计`、`纪实摄影`

## API 参数参考（火山方舟 / 豆包 Seedream）

> 来源：https://www.volcengine.com/docs/82379/1541523 及 API 文档

### 当前可用模型（2026年2月）

| 模型 | 说明 |
|------|------|
| `doubao-seedream-4-5-251128` | Seedream 4.5，最高分辨率，支持组图、流式输出 |
| `doubao-seedream-4-0-250828` | Seedream 4.0，稳定版本，支持组图 |
| `doubao-seedream-3-0-t2i-250415` | Seedream 3.0，支持 seed / guidance_scale |

### 关键请求参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `model` | string | 必填，模型名称 |
| `prompt` | string | 必填，提示词，建议 ≤ 300 汉字或 600 英文单词 |
| `size` | string | 图像尺寸，支持 `"2K"` / `"4K"` / `"宽x高"` 格式 |
| `seed` | int | 随机种子，-1 为随机（仅 3.0 支持） |
| `guidance_scale` | float | 提示词权重 1-10（仅 3.0 支持，默认 2.5） |
| `sequential_image_generation` | string | `"auto"` 开启组图 / `"disabled"` 单图 |
| `stream` | bool | 流式返回（逐图返回，适合组图） |
| `response_format` | string | `"url"` 或 `"b64_json"` |
| `watermark` | bool | 是否添加水印 |
| `optimize_prompt_options.mode` | string | 提示词优化：`"standard"` 或 `"fast"` |

### 尺寸速查（Seedream 4.5 / 4.0）

| 宽高比 | 推荐尺寸（4.5/4.0） | 用途 |
|--------|-------------------|------|
| 1:1 | 2048×2048 | 头像、方形图 |
| 16:9 | 2560×1440 | 横屏、PPT |
| 9:16 | 1440×2560 | 手机竖屏、短视频封面 |
| 4:3 | 2304×1728 | 传统横向照片 |
| 3:4 | 1728×2304 | 竖向照片 |
| 3:2 | 2496×1664 | 35mm 胶片横向 |
| 2:3 | 1664×2496 | 35mm 胶片竖向 |
| 21:9 | 3024×1296 | 超宽屏 |

> **4.5/4.0 自定义尺寸限制**：总像素 3,686,400 ~ 16,777,216，宽高比 1/16 ~ 16

### 组图（Sequential Image Generation）说明

- 触发词：`一系列` / `组图` / `生成N张连续的` / `分镜图`
- `sequential_image_generation: "auto"` + `max_images` 控制最大数量（上限 15）
- 参考图数量 + 生成图数量 ≤ 15

### API 调用示例（Python）

```python
import os, requests

url = "https://ark.cn-beijing.volces.com/api/v3/images/generations"  # 火山方舟
# 或使用代理: "https://www.dmxapi.cn/v1/images/generations"

headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {os.getenv('ARK_API_KEY')}"
}

payload = {
    "model": "doubao-seedream-4-5-251128",
    "prompt": "你的提示词",
    "size": "2K",                            # 或具体如 "2560x1440"
    "sequential_image_generation": "auto",   # 开启组图
    "sequential_image_generation_options": {"max_images": 6},
    "stream": False,
    "response_format": "url",
    "watermark": False,
}

response = requests.post(url, headers=headers, json=payload)
data = response.json()
for img in data["data"]:
    print(img["url"])
```

### 图片编辑（Seededit）调用示例

## Seedream 5.0 新特性（官方技术博客）

> 来源：https://seed.bytedance.com/zh/seedream5_0_lite

### 核心突破

1. **实时联网检索**（行业首创）
   - 动态获取最新外部信息，不受训练数据时间限制
   - 触发条件：提示词含时效词、热点词、长尾查询

2. **多步逻辑推理**
   - 理解抽象概念 → 拆解为可执行视觉指令
   - 示例：`"静谧科技感"` → 低饱和色调 + 简洁线条 + 冷光源 + 金属质感

3. **多轮对话式编辑**
   - 局部修改 / 风格迁移 / 元素增删 / 文字渲染 / 连续调整
   - 支持"像和设计师沟通一样"迭代优化

4. **高分辨率输出**
   - 原生 2K，AI 增强至 4K
   - 单图生成 2-5 秒

## 常见问题

**Q: 提示词多长合适？**
A: 官方建议 ≤ 300 汉字或 600 英文单词。过长会导致信息分散，模型可能忽略细节。

**Q: 英文还是中文提示词更好？**
A: 专业术语（摄影参数、艺术流派等）建议用英文；Seedream 4.0+ 的中文语义理解已大幅提升，日常描述用中文即可。

**Q: 如何让文字精准渲染？**
A: 将要生成的文字放入引号：`图片中央写着"Hello World"`

**Q: 如何触发联网搜索（5.0）？**
A: 在提示词中加入时效词：`2026年流行色` / `最新款XX` / `今年XX趋势`

**Q: 图片 URL 有效期多久？**
A: 返回的图片 URL 有效期 **24 小时**，请及时下载保存。

**Q: `seed` 参数有什么用？**
A: 固定 seed 值可复现相同构图，-1 代表每次随机。仅 Seedream 3.0 支持。
